# Vizat

**Vizat - A simple HTML5 Canvas Library**

<p>Version 1.0.7</p>
<p>by m3yevn</p>

<hr/>

 - Blazingly fast using tree structure to draw
 - Easy to configure with API similar to native HTML5 Canvas
 - Simple, Small and Concise

 - HTML5 Canvas API
